# siyeanwebsitesrstore

Source for the SR Mac Shop website.

**Primary project directory:** [`siyean-laravel/`](./siyean-laravel/) — this is
where you run Artisan, deploy from (`public/`), and run CLI helpers under
`scripts/`. The storefront/POS implementation lives in [`siyean/`](./siyean/)
and is loaded by Laravel through the legacy bridge.

See [`siyean-laravel/README.md`](./siyean-laravel/README.md) for local
development and [`deploy/cpanel/README.md`](./deploy/cpanel/README.md) for
cPanel.
